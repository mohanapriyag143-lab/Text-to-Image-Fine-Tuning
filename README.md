# Text-to-Image Fine-Tuning Projects

This repository contains internship projects related to text-to-image generation, GANs, attention mechanisms, dataset analysis, and text embeddings using deep learning techniques.

## Projects Included

### 1. Text-to-Image Fine-Tuning using LoRA
Fine-tuning a Stable Diffusion text-to-image model using LoRA techniques for efficient training and image generation.

### 2. Conditional GAN (CGAN) for Shape Generation
Implemented a Conditional GAN capable of generating shapes like squares, circles, and triangles based on text labels.

### 3. Oxford 102 Flowers Dataset Analysis
Performed dataset loading, preprocessing, visualization, and statistical analysis on the Oxford 102 Flowers dataset.

### 4. Text Embedding and Preprocessing
Used Hugging Face Transformers to tokenize and encode text descriptions into embeddings for text-to-image tasks.

### 5. Attention Mechanism in GANs
Implemented self-attention/cross-attention concepts to improve GAN image generation quality and focus on important text regions.

### 6. Complete Text-to-Image Pipeline
Built an end-to-end pipeline combining preprocessing, text embeddings, GAN-based image generation, and visualization.
